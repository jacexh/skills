# AI 历史反转全自动短视频工厂 - 设计文档

> **状态：** 已批准
> **创建日期：** 2026-03-04
> **负责人：** 一人公司创始人 (Xuhao)

## 1. 业务愿景
利用 AI 的高产出能力，通过“平行宇宙”和“历史反转”题材（赛道 2）构建自动化视频矩阵，通过平台播放量分成和广告变现实现被动收入。

## 2. 系统架构 (Technical Pipeline)
系统采用 Python 驱动的异步工作流：

### 2.1 脚本引擎 (LLM)
- **工具：** OpenAI GPT-4o API
- **输出：** 45-60秒旁白脚本 + 8-10个对应的视觉分镜 Prompt（英文）。

### 2.2 配音系统 (TTS)
- **工具：** ElevenLabs API (Voice: "Marcus" 或类似深沉解说音)。
- **输出：** MP3 音频文件 + 字幕时间戳 (JSON)。

### 2.3 视觉工厂 (Image Gen)
- **工具：** Fal.ai API (Flux.1 [dev] 模型)。
- **输出：** 8-10张 9:16 高清生成的静态图片。

### 2.4 自动化剪辑 (Assembly)
- **工具：** MoviePy (Python)
- **逻辑：** 
  - 图片按旁白节奏切换。
  - 应用 Ken Burns 缩放效果增加动态感。
  - 自动叠加带阴影的黄色/白色粗体字幕。
  - 随机选取预设的史诗级 BGM 进行混音。

## 3. 运营与变现
- **平台：** YouTube Shorts (主打), TikTok (副), Instagram Reels (副)。
- **频率：** 每日每个平台分发 2-3 条原创内容。
- **目标：** 3个月内跑通第一个频道的获利计划（Monetization）。

## 4. 成本预估 (单条视频)
- LLM: < $0.01
- TTS (ElevenLabs): \~bash.20 - $0.50
- Image Gen (Flux): \~bash.10 - $0.20
- **总计：** \~bash.31 - $0.71 / 条。
